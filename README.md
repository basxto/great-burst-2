## Great Burst 2

Continuation of Great Burst.

For [OGA Spring 2021 Game Jam [Special Edition!]][ogaspring]

Simple Break Out clone for Game Boy.

## Dependencies

* [GNU Make][]
* [SDCC][] 4.1.0
* [GBDK-2020][] 4.0.3

## Build

```
git clone --recursive https://github.com/basxto/great-burst2.git
cd great-burst2
make
```
